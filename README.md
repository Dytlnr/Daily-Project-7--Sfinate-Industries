## Deskripsi Sistem
SFINATE INDUSTRIES merupakan aplikasi berbasis web yang dirancang untuk membantu pengelolaan usaha sablon atau konveksi. Sistem ini menyediakan fitur utama seperti manajemen data barang, transaksi kasir, pengelolaan member dan poin, serta laporan penjualan. Aplikasi ini dikembangkan berdasarkan kebutuhan sistem yang telah dianalisis pada Daily Project 6, yang mencakup aspek fungsional dan non-fungsional seperti keamanan, performa, dan usability.

## Fitur Utama Sistem

### 1. Autentikasi & Akun
- Login dan logout untuk pengguna berdasarkan role (Admin, Kasir, Owner)
- Register akun baru
- Kelola profil pengguna (ubah data dan password)

### 2. Manajemen Barang
- Tambah, edit, hapus, dan lihat data barang
- Kelola jenis barang (kategori)
- Kelola barang masuk (stok dan supplier)

### 3. Transaksi Kasir
- Membuat transaksi pemesanan
- Menambahkan barang ke keranjang dengan validasi stok
- Menambahkan jasa sablon
- Menyimpan dan mengelola orderan
- Proses pembayaran (DP atau lunas)
- Cetak nota dan export PDF

### 4. Member & Poin
- Kelola data member
- Sistem poin berdasarkan transaksi
- Tukar poin member
- Riwayat poin (penambahan dan pengurangan)

### 5. Laporan
- Laporan penjualan
- Filter laporan berdasarkan tanggal dan status
- Export laporan ke Excel dan PDF
- Laporan piutang
- Laporan barang terjual
- Laporan pelanggan

### 6. Pengaturan & Keuangan
- Kelola data user
- Lihat data keuangan (pemasukan dan pengeluaran)
- Pengaturan poin
- Pengaturan identitas perusahaan
